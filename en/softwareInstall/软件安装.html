软件安装

一、安装管理程序​

1、安装管理程序（计费服务端程序）​

(1)左键双击"autorn"应用程序


(2)双击进入系统光盘页面后，点击“程序管理”---安装程序


(3)进入数据库类型选择界面，选择所需数据库后点击“下一步”

注：若单量不算很大，可选择“MySQL”数据库；若单量较大，
建议安装SQL Server数据库（SQL Server数据库需自备）


(4)默认用户名为：admin，密码：88888，选择好安装路径后，点击“安装”


(5)在如下弹窗中点击“完成”


(6)弹出一下弹窗后，即为程序安装成功

2、功能授权​

（1）安装成功，程序会自动打开，并弹出“软件授权”页面
（2）将“机器授权码”发送给“代理商相关工作人员”后工作人员会将
“授权码”发送给客户进行授权
（3）授权码填写完毕后，点击“授权”
（4）显示“授权成功”后进行重启，即服务端程序安装成功

二、安装Web查询系统​

1、IIS配置​

注意：安装WEB查询统计，根据电脑系统的不同，IIS安装配置方法就不同，请按照相应电脑系统先配置好IIS配置，再安装“WEB查询统计”（IIS安装方法可以根据系统版本点击跳转）

• Windows xp• Windows 7• Windows 8• Windows 2003
• Windows 2008• Windows 2012• Windows 2019

下面以 Windows 7 系统 IIS配置为例（其他系统请选择如上跳转链接进行配置）
Windows 7配置案例，Windows10\11同时适用

（1）设置Internet信息服务​

1）打开“控制面板”---“程序”---“打开或关闭Windows功能”---“Internet信息服务”

2）将Web管理工具和万维网服务二者进行勾选

3）Web管理工具设置如下图所示

4）在“万维网服务-安全性”中按照下图所示进行勾选

5）“万维网服务-应用程序开发功能”设置如下

（2）设置Internet信息服务（IIS）管理器​

1）控制面板---系统与安全---Windows工具---InternetInformation Services（IIS）管理器 ---“应用程序池”
相应的应用程序池会出现在右边
右键单击修改每个应用程序池，点击 “高级设置”修改

2）在启用32位应用程序在拉菜单中选择“True”，点击确定

2、安装Web查询系统​

(1)打开安装文件，左键双击“autorun”应用程序


(2)进入系统光盘界面，点击“Web查询统计平台”---再次点击“Web查询统计平台”--- 安装程序


(3)进入数据库类型选择界面，选择所需数据库后点击“下一步”
注：此数据库类型与安装“交换机通讯管理系统”数据库类型相同


(4)此界面数据库默认信息无需修改，点击下一步
  IP地址：127.0.0.1 用户名：root 密码：sa


(5)默认用户名为：admin，密码：88888，选择好安装路径后，点击“安装”

三、安装客户端​

1、安装中间件客户端​

（1）左键双击“autorun”应用程序


(2)双击进入系统光盘页面后，点击“中间件客户端”---安装程序


(3)进入数据库类型选择界面，选择所需数据库后点击“下一步”
注：此数据库类型与安装“交换机通讯管理系统”数据库类型相同


(4)此界面数据默认信息无需修改，点击下一步
  IP地址：127.0.0.1 用户名：root 密码：sa


(5)默认用户名为：admin，密码：88888，选择好安装路径后，点击安装

2、安装酒店版话务台客户端​

注：中间件客户端、酒店版话务台客户端、办公版话务台客户端，三者不可同时安装，只能安装一个版本，若想更换版本，请将原版本进行备份，以防数据丢失.备份后将原有软件进行卸载，重新安装其他版本。

点击“办公版话务台客户端”后，安装步骤与“中间件客户端”相同

3、安装办公版话务台客户端​

注：中间件客户端、酒店版话务台客户端、办公版话务台客户端，三者不可同时安装，只能安装一个版本，若想更换版本，请将原版本进行备份，以防数据丢失.备份后将原有软件进行卸载，重新安装其他版本。

点击“办公版话务台客户端”后，安装步骤与“中间件客户端”相同
